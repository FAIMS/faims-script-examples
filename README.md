# faims-script-examples

Collection of demos for accessing FAIMS3 data programmatically via the REST APIs.

Each language lives in its own subfolder with its own tooling and README.

## Examples

| Folder   | Description                                      |
| -------- | ------------------------------------------------ |
| [python](python/) | Read-only record fetch demo using long-lived API tokens |

## Getting a long-lived API token

1. Sign in to your FAIMS3 web dashboard
2. Open **Profile → Manage API Tokens**
3. Create a token and copy it immediately (it is only shown once)

See the [FAIMS3 long-lived tokens documentation](https://github.com/FAIMS/FAIMS3/blob/main/docs/developer/docs/source/markdown/Long-lived-tokens.md) for the exchange workflow and security guidance.
